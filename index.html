Q: What is Makaney Code?

Makaney code is a human-friendly alphanumeric representation (like postal or zip codes) of the geographic latitude and longitude coordinates. Using Makaney code helps easily identify places on earth with an easy to remember code. That's becomes very useful especially in countries that do not implement a practical postal code system. In such countries people tend to describe their locations using directions, which is not always straightforward, waste valuable time, and results in many inconveniences especially when an ambulance needs to reach a certain location as soon as possible.

Q: How do I get the Makaney code?

Just place the crosshair () in the center of the map at the desired destination and the Makaney Code will appear above (left your finger up -- for Makaney Code to be updated -- if using a finger touch device, e.g. iPhone).

Q: How do you calculate Makaney code?

The idea is very simple (and meant to be simple for ease of adoption), we take 4 decimal places of the decimal representation of the geographic latitude and longitude and convert them to Base 33 and then add the directions, which results in an easy to read, write, speak, and remember place location anywhere on earth.

Q: How accurate is Makaney Code?

Since Makaney Code is designed for civil services and is not meant to identify small objects, we sacrifice -- based on location -- few meters (not exceeding ± 5.55 m) of accuracy to make the code as short and easy to remember as possible.

Q: I am an application developer and my application would benefit from Makaney Code, how can I include Makaney Code to my software?

No need for a license, Makaney Code is a non-profit project, free for everyone to use. Feel free to add Makaney Code to your application to better serve your customers. Technical details are available here.

Q: Does GPS devices support Makaney Code to identify places?

Not until their customers request for Makaney code to be included. Including Makaney Code to GPS devices or any GPS application is very simple and straightforward and would make it very convenient for transportation. GPS providers would love to hear customers suggestions, write a line or two to your favorite GPS provider.

Q: My friends sent me their Makaney Code but my navigation system does not support the Code yet?

You can translate Makaney Code to geographic coordinates (latitude and longitude), that appear below Makaney Code in small brackets.
